إدارة الجامعة

بسم الله الرحمن الرحيم، الحمد لله رب العالمين وأفضل الصلاة وأتم التسليم على سيدنا محمد حبيب رب العالمين
أما بعد:
تتم إدارة الجامعة عبر كادر من الأساتذة الفضلاء أصحاب العلم والاختصاص.
يقوم أعضاء الجامعة من الإداريين والأساتذة بإدارة الجامعة والتدريس وإصدار الشهادات عبر موقع الجامعة.

الكادر الإداري

فضيلة الشيخ الطبيب: أسامة محمد سميح الأبرش (رئيس الجامعة)
فضيلة الشيخ الدكتور : ضاهر الحمود ( المدير الإداري ونائب رئيس الجامعة )
فضيلة الشيخ : عمار بازر باشي ( عميد كلية الشريعة والدراسات الإسلامية ورئيس قسم القرآن الكريم وعلومه )
فضيلة الشيخ : غياث قزموز (رئيس قسم الدعوة الإسلامية والدعاة )
فضيلة الشيخ المقرئ : عمار بازر باشي ( رئيس قسم القراءات )
فضيلة الشيخ الدكتور: موفق المحيميد ( رئيس قسم السيرة النبوية الشريفة )
فضيلة الشيخ الدكتور : غنى احمد عيواظة( رئيس قسم الفقه والفقه المقارن )
فضيلة الشيخ : محيي الدّين عبدالرّزاق صدّيق مزّاويّة ( رئيس قسم العقيدة الإسلامية ومقارنة الأديان )
فضيلة الشيخ الدكتور : حسين عبد الرحمن سليمان ( رئيس قسم الفقه وأصوله )
فضيلة الشيخ الدكتور: موفق المحيميد ( رئيس قسم الحديث النبوي الشريف وعلومه )
فضيلة الشيخ : محيي الدّين عبدالرّزاق صدّيق مزّاويّة ( رئيس قسم القضاء والإفتاء )
فضيلة الدكتورة: غنى احمد عيواظة ( عميد كلية اللغة العربية ورئيس قسم البلاغة والأدب )
فضيلة الشيخ : شفيق الدين الصالح ( رئيس قسم النحو والصرف )
